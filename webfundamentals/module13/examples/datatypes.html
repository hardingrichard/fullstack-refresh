Data Types

There are several data types in JavaScript. Here are a few examples:

  • String: "Hello, World!"
  • Number: 42
  • Boolean: true
  • Array: [1, 2, 3]
  • Object: {name: "John", age: 30}
  • Null: null
  • Undefined: undefined

These are just a few examples of the data types in JavaScript. There are many more, and you can learn more about them in the MDN Web Docs.
